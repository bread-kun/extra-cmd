# Extra command
This is a extension for expand text editor command. Add extra command with Deduplicate and Transpose. transposing selections or characters.

## Adding custom shortcuts
Add this to your `keybinding.json` file.
```javascript
{
    "key": "ctrl+alt+d",
    "command": "extension.deduplicate"
},
{
    "key": "ctrl+t",
    "command": "extension.transpose"
}
```

## Features
- swap selections
- remove all the deplicate selections
